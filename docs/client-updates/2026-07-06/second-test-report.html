Second Test — Results vs. Reference Calls

How to read this

We ran the system blind over 7 documents from 5 firms (BlackRock, Franklin Templeton, Goldman Sachs Asset Management, T. Rowe Price, and Wellington Management), then compared its output against a reference list of 89 calls that we prepared from our own reading of the same documents. The two will never line up perfectly, and that is expected. Where they differ, the differences fall into a few understandable categories, explained below.

Headline

Of the 89 reference calls, the system landed 69 on the exact same category as our reference list.

It captured a further 8 under an immediately adjacent category — the same view (overweight / neutral / underweight), just recorded against a neighbouring label. Counting those, the system covered the view on 77 of the reference calls.

Where both the system and our reference list made a call on the same category, they agreed on the direction 64 times out of 69 (about 93%).

Every supporting quote checked out against the source document — no call was kept without verified evidence behind it.

On the gap: a handful of the remaining reference calls (6) draw on firm material that sits outside the single linked document, and a small number were genuine misses that we have since addressed (see below). Setting aside the 6 that reference outside material, the system covered the view on 77 of 83 reference calls — about 93%.

Why the system produced calls the reference list doesn't have

The system recorded more calls than the reference list (145 in total). This is breadth, not error. Most of the extra calls come from faithfully reading a firm's full dial grid — every published dial becomes its own row — and from recording a view under an adjacent, more specific category than our reference list happened to use. Across the whole run, exactly one call was judged an over-read; the calls the system was unsure about were either capped in confidence or flagged for human review rather than asserted. How much of a full dial dashboard you want captured — every dial, or only a firm's highlighted views — is one of the questions on the questions sheet.

What we fixed as a result

Two of the firms publish their views as dial graphics on web pages that we capture as PDF, and the earlier pass had rejected some of those dial calls because it could only read the page as text. We tightened this: dial and chart graphics are now verified by the independent AI reviewer looking at the page image itself. Re-checking the affected documents this way, 22 of 23 previously-rejected dial calls cleared review, restoring coverage the earlier pass had dropped — including Wellington's Japan equities call, which now matches the reference list. We also tightened the wording rules so that language like "reduced back to neutral" lands on Neutral, and a two-sided argument ("out of one asset, into another") records both sides.

Two small points we have flagged for your confirmation. One reference call looks like a transcription slip: both dials in the source read Neutral, while the reference list records it as Underweight. And a few reference calls rely on a firm's broader publications beyond the single provided link — that point is on the accompanying questions sheet.
